# Video Segmentasyonu Tabanlı Futbol Karar Destek Sistemi

Bu proje, futbol maç videolarından nesne segmentasyonu gerçekleştirerek takım boyu ve eni gibi taktiksel metrikleri hesaplayan ve teknik heyete anlık karar destek önerileri sunan bir yazılım mühendisliği çalışmasıdır.

## 🛠️ Kullanılan Teknolojiler
* Python 3.x
* OpenCV (Görüntü İşleme ve Görselleştirme)
* NumPy (Matematiksel ve Matris Hesaplamaları)
* YOLOv8-Seg (Nesne Segmentasyonu - Simüle Edilmiştir)
